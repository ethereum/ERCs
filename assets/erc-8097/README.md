# ERC-8097 Reference Implementation

Reference implementation for [ERC-8097: In-Ground Asset Token](https://eips.ethereum.org/EIPS/eip-8097).

## Files

- `contracts/IERC8097.sol` — Interface (struct definitions + IERC8097)
- `contracts/ERC8097.sol`  — Reference implementation

## Install

```bash
forge install
```

## Test

```bash
forge test
```

## Deployment

This is a reference implementation only. Do not deploy to production without
an independent security audit and appropriate access control configuration.
